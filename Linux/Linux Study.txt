pwd : 현재 장업하고 있는 위치
※ / → 최상위 directory

ls : 현재 위치의 폴더에 있는 파일 보기
ls -F : 어떤 종류의 파일인지 확인
상세 옵션들은 다음 링크를 참조
https://www.yalco.kr/35_linux/

cd .. : 이전 폴더의 위치로 돌아가기
cd / : 최상위 directory로 가기
cd {원하는 파일 경로 -> 몇 글자 적고 tab누르면 자동 완성} : 파일로 이동

mkdir {원하는 디렉토리 이름} : directory 생성

vi {파일이름} : vim으로 파일 생성 및 에디터 열림
- i : 입력모드 전환
- esc : 현재모드 종료
- :wq : 저장 및 에디터 닫기
- :q : 에디터 닫기

cp {복사할 파일 이름} {새로운 파일 이름} : 복사할 파일을 새로운 파일에다가 복사

mv {옮길 파일 이름} {절대 혹은 상대 경로} : 해당 경로에 옮길 파일을 이동

rm {삭제할 파일 이름} : 파일 삭제
※ directory를 삭제하려면 -r 옵션을 붙여야 함

wget {다운로드할 것의 주소} : 웹에서 다운로드 할 경우
※ 마우스 오른쪽 버튼 : 붙여넣기 가능

sudo : 관리자 권한으로 실행
※ apt-get, apt : 데비안 계열의 리눅스 환경에서 소프트웨어들을 다운받아 설치하는 패키지 매니저
※ Ctrl + c : 뭔가 실행하고 있는 것을 종료

nano (파일이름)
less (파일이름)  enter q(빠져나가기)
grap (파일 일부) (파일이름)
